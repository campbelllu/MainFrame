Welcome to Investment Highlights!

A personal project made public for all to use.

The goal of every tool here is to create views for data that we wish we could find more easily.
What would we be looking for, what do we find relevant, most important? How can we get it all in one place?
So as our understanding of securities analysis grows, so too do our displays of data on this site.

Check the Navigation buttons up top to see what's available at this time.

The following is our planned order of updates that the site will receive over the coming months.

Database optimized, then updated for 2024 values. Estimated time to completion: End of March 2025.

This is paramount: The site is kind of silly to have if it doesn't have up-to-date 10-K data! Working with the data while making the UI taught us a lot about which parts of the DB were actually pertinent for use in analysis too. So a complete revamp of which data is stored and recalled will also be undertaken here.

UI Tweaks

This may become a constant process, but a reorganization of what data is presented on each page will be an upgrade process regardless.

Compare Stock Highlights. Estimated time to completion: Before EoY 2025.

We want to compare multiple securities in a way that doesn't involve multiple tabs of our site being open at once.

Dividend Stock Valuation. Estimated time to completion: Before EoY 2025.

The backend code works great... for us. Now we just have to get this put onto this site somehow. You want it, we got it, let's get it posted!

Graphical Representations of Data. Estimated time to completion: Before EoY 2025.

Dash, Plotly, Matplotlib, Tableau plugins, whatever. Let's get something up on here that shows the numbers in a way that human brains enjoy looking at.

AI.

That was our earnings call up above, did you see how it shot our stock price up 50% after reading it? Let's get some more tools on here that help project
possible outcomes for all this data. It doesn't have to be anything fancy, but playing with some machine learning models and presenting the data is
well within our wheelhouse of capability and would be fun for us at least. Worst case scenario it's just some linear regressions, plotted.

ETF Analysis Articles. Estimated time to completion: Ongoing
